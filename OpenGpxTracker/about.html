Open GPX Tracker v1.7.2

By Juan M. Merlos @merlos

& Vincent Neo @vincentneo

Created to help Open Street Map contributors to generate Public GPX traces and as a tool for developers to create real GPX files to test their apps.

Open GPX Tracker is an open source application distributed under GPL license. The source code is available on GitHub

Tip! Long press over the map to add a Waypoint.
